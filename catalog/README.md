# Catalog (generated)

`index.json` (this repository's own local, non-authoritative package index)
and `oasf.json` (a non-authoritative OASF-style discovery projection — see
the `ponytail:` note in `scripts/generate_oasf_projection.py`) are generated
by `scripts/validate_agents.py` and `scripts/generate_oasf_projection.py`.
Nothing in this directory is hand-edited, and the generated files are
gitignored. Run `make verify` to build them locally.
